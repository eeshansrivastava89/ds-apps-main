**New focus:** Main site = projects + unified analysis page (notebooks + posts). Writing = Substack (with links back).

# Phase 13: Charting Library Consolidation

> **Status:** Planned
> **Goal:** Replace Plotly + Leaflet with Apache ECharts to reduce bundle size by 12x and unify visualization API.

---

## Why This Change?

Current charting stack is bloated and split across two libraries:

1. **Plotly.js (3.5MB)** â€” Charts only (bar, distribution, funnel)
2. **Leaflet (300KB)** â€” Maps only (geo markers)
3. **Two separate APIs** â€” Plotly imperative vs Leaflet OOP
4. **Manual dark mode** â€” Separate theming for each library

**New approach:** Apache ECharts handles charts + maps in 300KB with unified API.
